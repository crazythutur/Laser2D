Detailed Description

Description of a state that caracterise the Markov Chain.

Author
Arthur Vallet
Version
1.0
Date
Date
02/07/2019

the state is caracterised by a number of variable that describe enterely the state. Each state is caracterised by the number of photon for each mode stocked in photons, the number of elctron in each band (Conduction and Valence bands) stocked in $ CB_electrons and VB_electrons.
