Detailed Description

extention of of the Marmote TransitionStructure For Laser simulation

Author
Arthur Vallet
Version
1.0
Date
Date
2019/01/03

it was composed princialyt by two MarmoteBox :

  • photon :represent the number of Photon for each mode
  • CB_electrons :reprensent the electron number in conduction band for all Emitter

MarmoteBox is a MarmoteSet with the state space rectangular MarmoteSet permit to reprensent a system with only a one dimention with the attribuation of a number for each state

then for simulate we need only one MarmoteSet so we use a carthesian product of the 2 previous MarmoteBox this value in stocked on set

then for being simulated By MarmoteCore the function TransDistrib is overwrited all the events are inside For using this object it should be used in the object MarkovChain then initialisated by the function initial_state Before being simulated.
