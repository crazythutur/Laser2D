Detailed Description

The Electron_presence class represent the probability of finding an electron at different level for a QW_Emitter for different number of electron in Conduction Band (CB) this probability is deffined for each lasing mode by by getLasing_lvl_prob([)CB_Electron_number, mode) or for pumping level by pumping_lvl_prob[CB_Electron_number] the mode_number energy_level_number represent the maximum electron capacity of the CB that correspond to the energy level number.
